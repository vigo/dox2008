  What is it?                                                                 
  ¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯  
  It is a simple Regular Expression tool for javascript. You can build your   
  own patterns, test them and save (manuelly) for further usage. It is full   
  of Html and Javascript code. Totaly free and public-domain. Feel free to    
  use/modify it. I suggest you to use with Firefox or other Mozilla based     
  browsers.                                                                   
                                                                              
  You set your own language via duplicating the file : “js/locale.js”. If     
  you wish to use it in German language, copy it like “js/locale_de.js” and   
  translate it by your self and change the line #21 at “index.html” like :    
                                                                              
  <script type="text/javascript" src="js/locale_de.js"></script>              
                                                                              
  Original release have 2 languages. English and Turkish. If you create any   
  other language, please send it to me! I love to include your work into the  
  release package!!!
